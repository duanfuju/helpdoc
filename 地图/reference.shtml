命名空间

AMap 是整个高德地图JavaScript API 的命名空间，本手册中介绍的所有类和对象都通过AMap.XXX 的方式调用。

如创建一个LngLat 对象：new AMap.LngLat(116.397428,39.90923)。

Map 类

地图基础类，地图实例化入口函数。

构造函数说明
AMap.Map(mapDiv:Node, opts?:MapOption) 地图基础类，地图实例化入口函数。
MapOption 类型 说明
level Number 地图初始化时的视野级别。
center LngLat 地图初始化时的中心点经纬度坐标。
zooms Array 地图视野级别范围，如：[3,19]，3 为最小级别，19 为最大级别。
defaultTileLayer TileLayer 地图初始化时默认的底图图层。
dragEnable Boolean 定义地图是否可以拖拽，此属性可被setStatus/getStatus 方法控制。
zoomEnable Boolean 定义地图视野级别是否可以更改，此属性可被setStatus/getStatus 方法控制。
keyboardEnable Boolean 定义地图是否可以被键盘控制，此属性可被setStatus/getStatus 方法控制；键盘方向键控制方向，键盘加减健控制视野级别缩放。
jogEnable Boolean 定义地图拖拽时是否支持缓动效果，此属性可被setStatus/getStatus 方法控制。
continuousZoomEnable Boolean 定义地图是否有连续缩放效果，此属性可被setStatus/getStatus 方法控制。
doubleClickZoom Boolean 定义地图是否可以双击放大，此属性可被setStatus/getStatus 方法控制。
scrollWheel Boolean 定义地图是否支持鼠标滚轮缩放，此属性可被setStatus/getStatus 方法控制。
crs String 坐标参考，取值：
'EPSG3857'
'EPSG3395'
'EPSG4326

setZoom(zoomLevel:Number) 设置地图视野级别。

getZooms( ) Array 获取地图当前视野级别范围。
getSize( ) Size 获取地图容器像素大小范围。
getBounds( ) Bounds 获取当前地图可视矩形区域的经纬度范围，即西南、东北角的经纬度坐标。
getTiles( ) Array 获取地图视野所有切片ID 的数组。
getStatus( ) Object 获取地图状态参数对象，如当前地图是否可拖动、放大等状态。
getResolution(point:LngLat) Number 返回指定坐标点处的分辨率，单位：米/像素。若指定参数 point，则返回该点的分辨率，否则，返回地图中心点的分辨率。
getScale(dpi:Number) Number 返回当前地图中心点的比例尺的倒数，即地图上单位长度代表的实际距离。 电子地图不同于纸质地图，其比例尺受屏幕分辨率影响，对于同一张地图，由于屏幕的分辨率可以调整，使用不同的屏幕，显示出来的比例尺可能不同。然而，无论屏幕分辨率如何调整，当图片固定时，图片的（空间）分辨率是不变的，因此，我们可以通过确定屏幕分辨率来确定地图的当前比例尺。
getDefaultCursor( ) String 获取地图默认鼠标指针样式。
zoomIn( ) 将地图视野放大一个级别。
zoomOut( ) 将地图视野缩小一个级别。

setBounds(bound:Bounds,

callback:Function) 		根据省份名称、城市名称、城市区号、城市邮编、区县名称、区县邮编进行位置定位与视野调整，当定位成功后执行回调函数。

TileLayer 类

切片图层类，该类为基础类。

构造函数说明
AMap.TileLayer(tileOpt:TileLayerOption) 构造一个切片图层对象。
TileLayerOption 类型 说明
id String 图层ID。
tileSize Number 切片大小，取值：
256，表示切片大小为256*256，
128，表示切片大小为128*128，
64，表示切片大小为64*64。
tileUrl String 取图地址。
zIndex Number 图层叠加的顺序值，0表示最底层。（自v1.1新增）
opacity Float 图层的不透明度，取值范围[0,1]，1代表完全不透明，0代表完全透明。（自v1.1新增）
getTileUrl Function(x,y,z) 获取取图地址函数。
reuseTiles Boolean 是否切片回收，默认false。（自v1.1新增）
detectRetina Boolean 是否区分高清屏，在touch 设备上，如果该值设置为true，则会根据手机屏幕分辨率，调用不同的地图图片，以适配不同屏幕分辨率。默认值是false。（自v1.1新增）
errorUrl String 取图错误时的代替地址。（自v1.1新增）
方法 返回值 说明
show( ) 显示图层。（自v1.1新增）
hide( ) 隐藏图层。（自v1.1新增）
setzIndex(index:Number) 设置图层叠加的顺序值。（自v1.1新增）
setOpacity(num:Number) 设置图层的不透明度。（自v1.1新增）
事件 返回值 说明
complete e 图层加载完成 。 （自v1.1新增）
